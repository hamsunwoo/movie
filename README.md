# mov

### install
```bash
# main
$ pip install git+https://github.com/hamsunwoo/movie.git

# branch
$ pip install git+https://github.com/hamsunwoo/movie.git@<BRANCH_NAME>
```

### start dev
```bash
$ git clone <URL>
$ cd <DIR> 
$ source .venv/bin/activate
$ pytest

#option
$ pdm venv create
```

### setting env
```bash
#MY_ENV
export MOVIE_API_KEY="<KEY>"
```
